PACOTE DE ASSETS — GUIA RÁPIDO

Pastas criadas:
- assets/image/  → imagens grandes (hero, stills)
- assets/thumb/  → miniaturas 16:9
- guides/        → guias (overlays) para composição

Substitua cada placeholder mantendo o MESMO nome do arquivo.
Sugestões de qualidade:
- WebP: qualidade ~82, “method 6”
- JPG (se preferir): qualidade 85–90

Export no Figma (simples):
1) Crie frames com as dimensões indicadas (Hero 1200×1500, Thumbs 800×450 e 1200×675).
2) Renomeie cada frame com o NOME-EXATO do arquivo destino (ex.: obra-01@800).
3) Selecione cada frame > Export > JPG ou PNG (ou use plugin para WebP) > substitua o placeholder.
